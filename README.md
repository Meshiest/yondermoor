# Yondermoor

### Getting Started

1. `git clone ...`
2. `npm install`
3. `gulp build` or `gulp watch` for continuous building
4. One of the following:
    * `python -m SimpleHTTPServer`
    * `npm install -g http-server` -> `http-server`
    * Open `index.html` in your browser

### Stack

* NPM
* Gulp
* React + Babel
* jQuery
* THREE
* WebRTC
* Sass
