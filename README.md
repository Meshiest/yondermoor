# Yondermoor

### Getting Started

1. `git clone ...`
2. `npm install`
3. `brunch build` or `brunch watch --server` for continuous building
4. View at `localhost:8000` if you chose to watch

### Stack

* NPM
* Brunch
* React + Babel
* jQuery
* THREE
* WebRTC
* Sass
